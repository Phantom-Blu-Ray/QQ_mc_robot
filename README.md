#### 使用说明

1. 手动修改json文件数据，在json里面添加服务器信息以及群号信息

2. 支持命令
```python
<管理>查看管理指令列表：/指令列表
<管理>查询机器人状态：/状态
<成员>测试机器人状态：/测试
<成员>查询服务器状态：/服务器状态
```
